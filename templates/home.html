Trabalho de Conclusão de Curso

Trabalho realizado para conclusão do Curso de Pós-Graduação em Desenvolvimento JAVA da Universidade Positivo.

Para execução deste trabalho foram utilizadas as seguintes tecnologias:

  • Java
  • Hibernate/JPA
  • Spring MVC
  • Spring Rest
  • AngularJS
  • HTML5 / CSS / JavaScript
  • jQuery
  • Bootstrap
  • Android Retrofit
  • Gson
  • Apache TomCat
  • Apache HTTP

O uso das tecnologias acima foram abordadas nos seguintes módulos:

  • Programação JAVA
  • Modelagem de Sistemas Orientados a Objetos
  • Design Patterns
  • Objetos Distribuídos
  • WebServices
  • Android - Serviços e Aplicações
  • Banco de Dados
  • Frameworks
  • Desenvolvimento Front End
